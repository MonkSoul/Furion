## 27.1.2 服务器环境配置

### 27.1.2.1 第一步

安装.NET Core 运行时捆绑包：[点击下载](https://dotnet.microsoft.com/permalink/dotnetcore-current-windows-runtime-bundle-installer)

### 27.1.2.2 第二步（命令）

```bash
net stop was /y
```

### 27.1.2.3 第三步（命令）

```bash
net start w3svc
```

### 27.1.2.4 第四步（命令）

```bash
set ASPNETCORE_ENVIRONMENT=Production
```
