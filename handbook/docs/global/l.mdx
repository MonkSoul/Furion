## 8.1 转换文本多语言

```cs showLineNumbers 
var apiInterface = L.Text["API 接口"];
```

## 8.2 转换 Html 多语言

```cs showLineNumbers 
var name = L.Html["<b>Hello</b><i> {0}</i>", name];
```

## 8.3 设置当前语言

```cs showLineNumbers 
L.SetCulture("en-US");
```

## 8.4 获取系统语言列表

```cs showLineNumbers 
var list = L.GetCultures();
```

## 8.5 获取当前选中语言

```cs showLineNumbers 
var list = L.GetSelectCulture();
```
